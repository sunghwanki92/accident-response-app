# 🚨 교통사고 대응 앱

대한민국 교통사고 발생 시 현장에서 바로 사용할 수 있는 올인원 웹 앱입니다.

## 주요 기능

- 📍 **현장기록** — 5단계 사고 기록 (GPS 자동 위치, 사진/영상 첨부)
- 🤖 **AI 분석** — 과실비율 분석, 실시간 법령 검색
- 📄 **서류 발급** — 교통사고 경위서, 보험 청구서 자동 생성 및 PDF 저장
- 🛡️ **보험사 대응** — 보험사 발언 AI 분석, 대응 전략 제공
- 📑 **보험 분석** — 보험 증권 PDF 업로드 → 보장 한도 분석
- 👤 **내 정보** — 차량 다중 등록, 보험사 자동 입력
- ⚙️ **설정** — 사고 기록 목록 관리, PDF 저장, 이전 기록 비교

## 기술 스택

- Vanilla HTML/CSS/JavaScript (단일 파일)
- Anthropic Claude API (AI 분석)
- OpenStreetMap Nominatim (GPS 역지오코딩)
- LocalStorage (사고 기록 저장)

## 사용 방법

`index.html` 파일을 브라우저로 열거나 GitHub Pages로 배포하세요.
